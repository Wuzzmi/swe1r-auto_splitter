# Star Wars Episode I Racer Autosplitter

Automates the LibreSplit timer, for Star Wars Episode I Racer speedruns.

### Features

* Auto Start, Split, and Reset (based on settings / catagory)
* In script settings
    - Presets for run catagory
    - and/or advanced full settings control
* Option to view extra information in terminal (when LibreSplit is run through terminal)

### Requires

* A functioning installation of the re-released PC versions of Star Wars Episode I Racer(Steam, GOG, etc.) and won't work with the original CD version.
* [LibreSplit](https://github.com/LibreSplit/LibreSplit/tree/main)
* Text editor

### Setup

* Download and open the autosplitter file (swe1r-autosplitter.lua).
* Scroll down until you see the commenting that says "AUTOSPLITTER SETTINGS". Here is where all the available settings are edited.
